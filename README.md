# 🐍 Python Beginner DSA Projects

Welcome to my collection of small **Python-based projects** that help me practice and understand **Data Structures & Algorithms (DSA)** and core programming concepts. These are beginner-friendly and written with simplicity in mind 😊

---

## 📂 What's Included

| 🧪 Project Name            | 💡 Description                                     |
|---------------------------|----------------------------------------------------|
| `countVowels.py`          | Counts how many vowels are in a given string       |
| `reverseString.py`        | Reverses a string using a loop                     |
| `palindrome.py`           | Checks if a string is a palindrome                 |
| `replace_spaces.py`       | Replaces all spaces with `_` in a string           |
| `numberguessinggame.py`   | Basic number guessing game with input              |
| `numberguessGUI.py`       | GUI version of number guessing (Tkinter) 🎮        |
| `pracctice.py`            | (Misc) file for extra learning/testing             |
| `hehe.py`                 | Just me being playful while learning 😄            |

## 🛠️ How to Run

1. Clone this repo or download the ZIP
2. Make sure Python is installed
3. Open terminal and run:
```bash
python filename.py
```
